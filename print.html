Rust

Rust was the first compiled programming language I learned about 5 years ago and I'm still using it to this day. Starting out, I had some knowledge about memory management. I knew that some languages had garbage collection while others preferred manual memory management. Relying on a language to handle memory for me would result in less control than I wanted. Manual memory management felt quite daunting because I knew that could easily lead to a number of unfortunate mistakes. Rust gives me both the control and safety I desire at the same time. It's syntax also seems to fit fairly well within the way I happen to think.

fn main() {
    println!("Hello there.");
}

Webify

Background

This project started from a desire to share media over the lan. Over time, more features were added as I had more ideas and my knowledge grew. Now, this project represents my current best attempt at creating a portable web service in Rust.

Features

  • It's just a single exe file.
  • Generates default files if config is missing.
  • Modular routes to specific files.
  • SSL with dual-protocol support.
  • Inserts media from local directory on a web page with self-made templating code.

Configuration

Configuration is done with a config.toml file. The default config contents can be seen below.

scope = "localhost"
ip = "192.168.1.142"
port = 12345
ssl_enabled = false
ssl_port = 12345
ssl_cert_path = "pems/cert.pem"
ssl_key_path = "pems/key.pem"
todo_enabled = true
todo_scope = "localhost"
todo_ip = "192.168.1.142"
todo_port = 11111
upload_size_limit = 2000000000 # 2 GB (2 * 1000 * 1000 * 1000)
#upload_size_limit = "disabled" # disables the limit and allows any size (not recommended)
upload_storage_limit = 2000000000
browser = false

[routes]
"/" = ["static/home.html"]
"/stuff" = ["static/stuff.html", "static/media"]
"/pdf" = ["static/pdf.html", "static/documents"]
"/downloads" = ["static/downloads.html", "static/files"]
"/playlists" = ["static/playlists.html", "static/audio/"]
"/upload" = ["static/upload.html"]
"/files" = ["static/uploads.html", "uploads"]
